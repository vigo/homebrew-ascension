# Homebrew Tap for Ascension

ANSI/ASCII art viewer and text editor for macOS.

## Installation

```bash
brew tap vigo/ascension
brew install --cask ascension
```

## Update

```bash
brew upgrade --cask ascension
```

## Uninstall

```bash
brew uninstall --cask ascension
```
